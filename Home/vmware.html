VMware Fusion/Workstation

VMWare Fusion and Workstation are available for all RIT students through the RIT Home Use Portal.

Need to run Windows on your Mac, see below for instructions.

VMware Fusion on a Mac

Start by downloading VMWare Fusion from the RIT Home Use Portal. Then download Windows 10 from Azure Dev Tools for Teaching.

System Requirement

  • Minimum 6GB of RAM
  • Minimum 256 GB of Disk Space (SSD Recommended)

Some Macs, like the MacBook Air, are commonly sold with a 128gb SSD and 4gb of RAM. This is NOT sufficient to virtualize a copy of windows 10 and the hard drive is too small to bootcamp the system. In this instance, it is recommend to virtualize Windows 7 on MacBook models that cannot meet the above specification.

Installation Instructions

  • Mount the VMware Fusion DMG file you downloaded from the RIT Home Use Portal.

  • Open the Vmware Rusion installer and run the installer.

  • Use the license key from the RIT Home Use Portal that was provided when downloading VMware Fusion.

  • Launch VMware Fusion
  • Walk through the Setup Virtual Machine Wizard.

  • When prompted for an .iso file of the operating system, attach the one you have downloaded from the Azure Dev Tools for Teaching.

  • Leave default RAM and performance limits as is.
  • Finish Wizard.
  • Launch new virtual machine
  • Follow windows setup instructions.
  • Voila! You now have a Windows virtual machine installed on your Mac. You may now install any software you need on the Windows system.
